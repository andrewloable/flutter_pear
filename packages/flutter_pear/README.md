# flutter_pear

Dart-idiomatic Flutter API for the [Pear](https://pears.com/) P2P stack over a
bundled [Bare](https://github.com/holepunchto/bare) worklet. See the
[repository README](../../README.md) for the overview and quick start.

> Early / pre-release. M0 (worklet echo) is scaffolded; PearSwarm and friends are
> shaped but not yet backed by a real worklet.
