# flutter_pear

Dart-idiomatic Flutter API for the [Pear](https://pears.com/) P2P stack over a
bundled [Bare](https://github.com/holepunchto/bare) worklet. See the
[repository README](../../README.md) for the overview, install instructions,
and quick start.

> Pre-1.0, published on pub.dev. The Bare Kit worklet is real (not a
> stand-in), and every data-structure wrapper (Corestore/Hypercore, Hyperbee,
> Hyperdrive, Autobase, blind pairing) is implemented and fake-tested
> end-to-end, with real-worklet validation on a real Android emulator — see
> the repository README's "What works today" section for what's left before
> a 1.0 (physical two-device hardware validation and iOS are still ahead).
